# Online Retail Recommendation System

This project analyzes customer purchase behavior using an Online Retail dataset and builds a simple product recommendation system using machine learning.

## Tools & Technologies
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Technique
Collaborative Filtering using Cosine Similarity

## Project Steps
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Sales Trend Analysis
4. Country-wise Sales Analysis
5. Product Recommendation Model

## Dataset
The dataset contains online retail transactions including:
- Invoice Number
- Product Description
- Quantity
- Invoice Date
- Unit Price
- Customer ID
- Country

Note: Dataset is not included in this repository due to size limitations.
